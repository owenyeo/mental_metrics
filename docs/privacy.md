# Privacy Note

## Data collected

- Pseudonymous user identifiers
- Provider project identifier
- Timestamped interaction events
- Operational service metadata such as tier, intervention type, and referral destination

## Data to avoid

- Names, emails, phone numbers, addresses
- Free-text symptom narratives
- Clinician notes
- Message transcripts

## Consent considerations

Real deployments should align event tracking with consent, youth safeguarding policy, applicable health/privacy regulations, and provider retention requirements. This MVP assumes providers determine lawful basis and present appropriate consent notices where required.
